      What Pulse collects

      • The name of your active application, mapped to a category (e.g. "Visual Studio Code" → development). The app name only — never the window title.
      • Active vs. idle time (idle = no input for over 3 minutes), measured as seconds since your last input — never the input itself.
      • Focus-block timing (runs of 25+ minutes of uninterrupted productive activity).
      • Focus minutes per hour of your local day.
      Only the aggregated daily summary leaves this machine — never raw per-app events. Meeting minutes and task counts are part of the data contract but are not collected yet (sent as 0 until calendar/PM integrations).

      What Pulse never collects

      • keystrokes
      • screen contents / screenshots
      • window titles
      • URLs or browsing history
      • message or document contents
      • mouse-movement logging
      • webcam / mic
